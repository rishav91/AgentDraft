AgentDraft is a CLI- and canvas-first builder for AI agents, running on LangGraph. Every agent is defined as a declarative schema rather than hand-written graph code, and a compiler translates that schema into a real, runnable LangGraph StateGraph. The schema, CLI, canvas, and meta-agent layers only ever interact with an execution-backend interface — never with LangGraph directly — so LangGraph is a swappable implementation behind that interface rather than a hard dependency baked into the rest of the system. The concrete second target for that interface is AgentWeave, a lightweight custom agent SDK, which becomes an alternate backend once it's stable enough to support one.
Capabilities like memory, durability, sandboxed tool execution, and tracing are owned and implemented by whichever backend is running the agent, not by AgentDraft itself — but AgentDraft still integrates with all of them directly. Its schema exposes fields to configure memory, persistence, and sandboxing declaratively; its backend adapters translate that configuration into each backend's native mechanisms (checkpointers, sandboxed tool wrappers, execution traces); and its CLI, canvas, and trace viewer expose, control, and visualize that behavior — including actions like resuming a crashed run — by calling through the adapter rather than reimplementing any of it. AgentDraft also validates schemas against the capabilities of the selected backend, surfacing clear errors when a requested feature isn't supported rather than failing silently at runtime. Skills and MCP-based tool access are treated as a shared, backend-neutral capability layer that both AgentWeave and the LangGraph adapter draw from, rather than being implemented separately in each.
The build is sequenced from most to least deterministic: first a CLI for defining and running agents by hand-authoring schema files, then a visual canvas built on the same schema for designing and inspecting agent graphs, and finally a meta-agent — itself an agent built on this system — that generates and iteratively refines agent schemas from natural-language descriptions.
